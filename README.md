# leptitcafe
Projet Web SAE - Le P'tit Café - BUT MMI

# Guide pour l'équipe Front --->

## Où vous pouvez travailler ?
- HTML : `app/Views/pages/` et `app/Views/layouts/`
- CSS : `assets/css/`
- JS : `assets/js/`
- Images : `assets/images/static/`

## Ne touchez PAS à :
- `app/Controllers/`
- `app/Models/`
- `api/`
